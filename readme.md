### LORDLESS Smart Contracts
This repo is the smart contract source code for the LORDLESS game.

## Development 
   
```sh

npm install 
# or
yarn install


# test
# start your testrpc
npm run test

# deploy 
truffle deploy --network <network> --reset 

```
   
[host]: Web Service API
